# セキュリティキーっていいよね

セキュリティキーがユーザー体験の観点で大変気に入っております。

なおここで言うセキュリティキーというのは、FIDO Allianceが定めるところのFIDO Security Keysと考えてください。

# 小難しい認証鍵運用

最近の認証鍵運用は市民にとって小難しいと感じています。

パスワードマネージャーを使えなければならなかったり、最近は認証アプリなるものを使わなければならなかったり、そのUXが分かりやすいとはとても言えません。

# シンプルなUX

鍵が実体としてあること、これって素晴らしいです。鍵を他人に渡さないこと、この運用の分かり易さがよくわからないインターネットの世界からユーザーを守ります。

実際にセキュリティキーをいくつかのサービスにて使ってみましたが、登録使用共に必要な操作は多くないと感じました。

# 鍵は自分だけが持っている

鍵が自分の手元にだけあり、他の誰にも共有されていないこと。これも素晴らしいです。

パスワードマネージャーやクラウドサービスを介して企業が鍵を覗くリスクもないし、突然のサービス停止で鍵が使えなくなってしまうリスクもないです。

# パスキーよりいいかも

Multi-Device FIDO Credentials、Passkeysとも呼ばれていたものですが、最近はこちらがパスワードに替わる良い認証鍵と知れ渡りつつありますね。

ですが私はセキュリティキーの方が好きかも。
